# Announcing Bloc Hub

Running large language models locally has evolved rapidly, but compiling, optimization, and dependency configurations are still highly fragile. Developers often spend hours wrestling with CUDA versions, llama.cpp makefiles, and tokenizers just to run a model.

We built **Bloc Hub** to solve these issues. Bloc acts as a portable image registry for local AI runtimes, enabling you to package:
- Quantized GGUF weights
- Custom engine configs (threads, GPU context, flash attention)
- Reproducible environments

## The Single-Command AI Stack

Anyone on your team can now pull and launch the identical local-first setup in seconds:

```bash
bloc deploy arnav/qwen-3.5-9b-super
```

This is only the beginning of collaborative, decentralized AI infrastructure. By removing the developer friction of local model management, we hope to enable standard, reproducible AI environments.
